# Auth Module

## Installation

To install, add the following to composer.json file

or add

```json
{
  "require": {
    ***
   "codexten/yii-module-auth": "~2.0"
  },
  "extra": {
    "config-plugin": {
      ***
      "???": [
        "$moduleAuth",
        "config/???.php"
      ]
  }
}
}

```
