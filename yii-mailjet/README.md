# Mailjet

## Installation

To install, add the following to composer.json file

or add

```json
{
  "require": {
    ***
   "codexten/yii-mailjet": "~2.0"
  }
}
```

## params

```php
return [
    // default mailer
    'mailer' => 'mailjet',
        
    //mailjet
    'mailjet.apikey' => '',
    'mailjet.secret' => '',  
];


```
